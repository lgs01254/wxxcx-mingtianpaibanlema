# 明天排班了吗

一款简洁高效的微信排班小程序，支持单人排班、多人管理和数据分享。

## 功能介绍

### 📅 单人排班
- 日历式排班界面，点击日期即可选择
- 支持多种班次类型：早班、中班、休息等
- 自动统计各类班次天数
- 支持从Excel导入排班数据
- 清空排班一键操作

### 👥 多人排班管理（管理员面板）
- 员工名单管理：添加、编辑、删除员工
- 批量导入员工排班（支持从微信聊天记录或本地文件选择Excel）
- 支持工作表选择，自动识别排班数据
- 批量删除员工功能
- 分享排班给微信好友

### 📤 数据分享
- 单人排班分享：分享当月排班给好友
- 全部员工排班分享：一次性分享所有员工排班
- 收到分享后可选择保存全部排班或单独加载某个员工数据

### 📊 Excel导入
- 支持 .xlsx、.xls、.csv 格式
- 智能识别表头和日期列
- 自动识别员工姓名和排班数据
- 多工作表支持，可自由选择

## 页面结构

```
pages/
├── index/          # 主页（单人排班+分享）
├── schedule/       # 单人排班详细页
├── settings/       # 设置页
├── shift-manage/   # 班次管理
└── admin/          # 管理员面板（多人排班管理）
```

## 技术栈

- **框架**：微信小程序
- **开发语言**：JavaScript
- **UI**：WXSS + WXML
- **数据存储**：本地存储（wx.setStorageSync）
- **Excel解析**：xlsx库

## 使用说明

### 1. 单人排班
1. 进入首页，点击日期进行选择
2. 选择顶部班次类型
3. 点击"保存排班"确认

### 2. 多人排班管理
1. 进入"管理员面板"
2. 点击"导入Excel排班"选择文件
3. 选择工作表和员工后确认导入
4. 点击员工姓名可进行分享、编辑、删除等操作

### 3. 数据分享
1. 单人模式：点击右上角"分享"按钮
2. 管理员模式：点击员工姓名，选择"分享排班"或使用"分享全部员工排班"
